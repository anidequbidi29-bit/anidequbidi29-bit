# Hi, I'm building ImgPrompts

I'm a developer and indie site builder working on web products around AI-assisted visual creation.

My current project is [ImgPrompts](https://www.imgprompts.net), a web-based workspace for AI image prompts, image generation, and image editing. It helps creators, designers, marketers, and product teams turn rough visual ideas into reusable prompts and more consistent image workflows.

## What I'm Working On

[ImgPrompts](https://www.imgprompts.net) is designed for people who need to create, refine, and reuse visual ideas across different projects.

The product focuses on:

- Structuring image ideas into clear, reusable prompts
- Supporting AI image generation and editing workflows in one web workspace
- Helping creative and marketing teams keep visual direction more consistent
- Making prompt iteration easier to manage over time

## Why I Build This

AI image tools are powerful, but the workflow around them can become scattered quickly: prompts live in notes, references are hard to reuse, and small edits often require starting over.

ImgPrompts is my attempt to make this process more organized, repeatable, and useful for real creative work.

## About Me

I'm interested in AI products, visual workflows, SEO-driven web projects, and practical tools that help people create faster without losing control over quality.

I care about building products that are:

- Clear in purpose
- Useful in everyday work
- Simple enough to adopt
- Flexible enough to grow with users' workflows
